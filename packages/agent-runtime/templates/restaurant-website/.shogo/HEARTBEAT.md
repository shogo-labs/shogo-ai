# Heartbeat

Heartbeat is **off** by default for this template — a website doesn't need a
background loop. Turn it on (set `heartbeatEnabled: true` in `config.json`)
only if the owner wants proactive checks such as:

- A daily summary of new table bookings and contact-form messages.
- A reminder to update seasonal hours (holidays, long weekends).
- A weekly nudge if the gallery or menu still looks empty.

When enabled, keep it quiet and useful — one short digest, not noise.
