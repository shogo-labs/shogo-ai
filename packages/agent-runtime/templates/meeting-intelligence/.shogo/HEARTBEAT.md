# Heartbeat Tasks

## Every 30 Minutes (Active Hours)

- **Scan upcoming meetings** — Check Google Calendar for meetings starting in the next 2 hours; flag any without prep notes
- **Trigger prep for new meetings** — If an unprepped meeting is found, run the meeting-prep-v2 skill automatically

## Every 2 Hours

- **Refresh research** — For meetings later today or tomorrow, check for breaking news on attendee companies
- **Update schedule canvas** — Reflect any new invites, cancellations, or time changes from calendar

## Daily (Morning)

- **Build daily briefing** — Compile a full day view: all meetings, attendee summaries, and key talking points for each
- **Surface open action items** — Pull up any unresolved follow-ups from the past week and flag overdue items

## Daily (Evening)

- **Prompt for meeting capture** — Ask the user if there are any meetings from today that need summaries or action items logged
- **Update relationship memory** — Save any new context about contacts encountered today
