# Dcf Valuation

Build a valuation memo with explicit revenue, margin, WACC, terminal value, and sensitivity assumptions for one company.

## Required Inputs
- User objective and time horizon.
- Relevant tickers, sectors, holdings, or macro topics.
- Risk tolerance and account context when the output affects a portfolio.
- Source links or permission to research current data.

## Process
1. Clarify missing inputs that would materially change the result.
2. Gather or request data from reliable sources.
3. Separate facts, assumptions, estimates, and opinions.
4. Produce a concise memo with source citations and confidence labels.
5. Persist durable outputs through the Prisma-backed API for `Equity Research Terminal`.

## Output Rules
- Never present unsourced figures as facts.
- Do not give personalized financial advice or instructions to trade.
- End with open questions, risks, and suggested next research steps.
