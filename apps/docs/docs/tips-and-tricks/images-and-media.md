# Images & Media

Learn how to add images, icons, and media to your Shogo apps through chat prompts.

## Adding images

### Using image URLs

The simplest way to add images is by providing a URL:

> "Add a hero image at the top of the landing page using this URL: https://example.com/hero-image.jpg. Make it full-width with a max height of 400px."

### Placeholder images

If you don't have final images yet, ask for placeholders:

> "Add placeholder images to each product card. Use a gray background with the text 'Product Image' centered."

> "Add profile avatar placeholders — circular, 48px, with the user's initials."

### Image galleries

> "Create an image gallery on the Portfolio page that shows images in a 3-column grid. Clicking an image should open it in a larger view."

## Icons

### Using icon libraries

Ask the AI to use popular icon libraries:

> "Add icons to the navigation: a home icon for Dashboard, a people icon for Clients, a folder icon for Projects, and a gear icon for Settings."

> "Add a search icon inside the search input field."

### Status indicators

> "Add colored status dots next to each project: green for Active, yellow for On Hold, red for Overdue, and gray for Completed."

## Layout with images

### Hero sections

> "Create a hero section with a background image, a centered headline in white, a subtitle, and a call-to-action button."

### Cards with images

> "Each product card should have an image at the top taking up 60% of the card height, with the product name, price, and description below."

### Team/profile sections

> "Create a team section with profile cards in a row. Each card has a circular photo at the top, the person's name, their role, and social media links."

## Design tips

:::tip Be specific about sizes
When adding images, specify dimensions or constraints:
> "The logo should be 120px wide. Product images should be 200x200px, cropped to fit."
:::

:::tip Consider mobile
Images that look great on desktop may need different treatment on mobile:
> "On mobile, show one product card per row with the image smaller. On desktop, show three per row."
:::

:::tip Use consistent styling
For a polished look, apply consistent image treatment:
> "All images in the gallery should have rounded corners (8px) and a subtle shadow. Maintain a consistent aspect ratio of 16:9."
:::

## Background images and patterns

> "Add a subtle gradient background to the header section — from dark blue on the left to medium blue on the right."

> "Use a light gray background with a subtle dot pattern for the main content area."

## Media embedding

### Videos

> "Add a product demo video to the homepage. Embed it in a centered container with rounded corners and a play button overlay."

### Maps

> "Add a Google Maps embed showing our office location on the Contact page. Make it 400px tall and full-width."

## FAQ

**Can I upload images directly?**
You can attach images to your chat messages. The AI can use them as reference for design decisions. For images displayed in your app, use URLs pointing to hosted images.

**What image formats work?**
Standard web formats: JPEG, PNG, GIF, SVG, and WebP.

**Can I add a file upload feature?**
Yes! Ask the AI:
> "Add a file upload field where users can upload an image. Show a preview of the uploaded image."
